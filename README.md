# 📝 Ojae Editor

[👉 웹에서 바로 사용해보기](https://choe050919.github.io/ojae-editor/)

**Ojae Editor**는 소설 및 장문 집필을 위해 설계된 **웹 기반 클라우드 텍스트 에디터**입니다.

별도의 설치 없이 브라우저만 있으면 어디서든 글을 이어 쓸 수 있으며, 섹션 관리와 BGM 플레이어 등 작가에게 필요한 편의 기능을 제공합니다.


## ✨ 주요 기능 (Features)

### 1. 챕터 및 섹션 관리

* **섹션 분리:** 긴 글을 여러 섹션(회차)으로 나누어 관리할 수 있습니다.
* **순서 변경:** 드래그 앤 드롭으로 섹션의 순서를 자유롭게 변경할 수 있습니다.
* **노트 모드:** 섹션을 '본문' 또는 '노트' 타입으로 설정하여 설정집이나 아이디어 메모를 원고와 함께 관리할 수 있습니다.

### 2. 강력한 집필 환경

* **실시간 자동 저장:** Firebase 기반으로 작성 중인 내용이 실시간으로 클라우드에 저장됩니다.
* **고유 URL:** 생성된 문서는 고유 URL(`/#docId`)을 가지며, 링크만 있으면 어디서든 다시 열어볼 수 있습니다.
* **목표 글자 수:** 일일 목표 글자 수를 설정하고 진행 상황을 시각적으로(프로그레스 바) 확인할 수 있습니다.
* **집중 모드:** 집필에 방해되는 UI 요소를 모두 숨길 수 있습니다.
* **다크/라이트 모드:** 눈의 피로를 줄여주는 테마를 지원합니다.

### 3. 멀티미디어 & 유틸리티

* **유튜브 미니 플레이어:** 에디터 내장 플레이어로 BGM이나 자료 영상을 보며 작업할 수 있습니다 (창 크기 조절 가능).
* **마크다운 미리보기:** 작성한 글을 읽기 모드(Markdown 렌더링)로 전환하여 가독성 있게 검토할 수 있습니다.
* **백업 및 내보내기:**
* 전체 프로젝트 또는 개별 섹션을 `.json` (복구용) 또는 `.txt` (텍스트 파일)로 내보내기/불러오기 할 수 있습니다.


## 🛠 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3 (Variables, Grid/Flex), Vanilla JavaScript
* **Backend (BaaS):** Firebase Realtime Database (실시간 동기화)
* **Libraries:**
* [Marked.js](https://github.com/markedjs/marked): 마크다운 렌더링
* [Phosphor Icons](https://phosphoricons.com/): 아이콘 팩
